# M3Fund - Site Statique

Site web statique de présentation pour la plateforme M3Fund - Plateforme de Crowdfunding au Mali.

## Structure

- `index.html` - Page principale du site
- `styles.css` - Feuille de style CSS
- `script.js` - Script JavaScript pour les interactions
- `crow.jpg` - Image de présentation

## Couleurs

- Vert principal : `#06A664`
- Jaune doré : `#E6B800`
- Blanc : `#FFFFFF`
- Gris foncé : `#2D2D2D`

## Fonctionnalités

- Design responsive (mobile, tablette, desktop)
- Navigation fluide avec scroll smooth
- Animations au scroll
- Compteurs animés pour les statistiques
- Header sticky
- Sections : Hero, À propos, Fonctionnalités, Comment ça marche, Types de projets, CTA, Footer

## Utilisation

Ouvrez simplement `index.html` dans un navigateur web pour visualiser le site.

## Déploiement

Le site peut être déployé sur n'importe quel serveur web statique (GitHub Pages, Netlify, Vercel, etc.).
